# Changelog

All notable changes to the Mycelium Memory Hub will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-02-10

### Added
- Core Memory Hub server with Express + Socket.IO
- SQLite database for development, PostgreSQL for production
- REST API for conversations, projects, patterns, and sessions
- Context Manager with cross-platform context building
- AI Visitor Tracker middleware for request logging
- Project Scanner for auto-discovering local projects
- Platform bridges (VS Code, Web Chat)
- External bridge manager for cross-service integration
- Mycelium Network bridge for real-time AI-to-AI communication
- MCP server implementations (Memory Hub + Mycelium Network)
- Federation Hub v2 with 22 services
- Docker support with health checks
- Fly.io deployment configuration
- OpenAPI and AsyncAPI specifications
